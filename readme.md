# puffy-pocket

## requirements

node, npm

## get started

- git clone
- run `npm i` to install dependencies
- copy `src/config/config.example.ts` to `src/config/config.ts` and edit parameters

## start

- `npm start` for production mode
- `npm run dev` for development mode
